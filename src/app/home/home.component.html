Accueil

Ce site web a été créé dans le but de servir de référence pour les cours LOG2990 et LOG4420 à Polytechnique Montréal. Il regroupe une panoplie d'informations utiles qui peuvent vous aider à réaliser votre projet intégrateur ou vos travaux pratiques.

La section « Tutoriels » regroupe plusieurs tutoriels sur les technologies utilisées dans les cours, afin de faciliter votre apprentissage de celles-ci. La section « Guide », elle, réunit un bon nombre de bons exemples et de contre-exemples concernant l'utilisation des technologies web. Il constitue en quelque sorte un guide de bonnes pratiques pour le développement web.

Dans l'optique d'amener un esprit de collaboration et de transparence, l'ensemble du contenu du site web est disponible sur notre page GitHub.
